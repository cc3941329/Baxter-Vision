How setup and start Baxter

* Краткий мануал по запуску:

* Подсоединить Клаву

* Запустить через кнопку на сзади

* Нажимать Alt+F, F пока не появиться окно консоли с выбором запуска

* Applications -> RSDK GUI

* sudo su (password: 1)

* avahi-daemon

* Далее зайти в baxter

* Выполнить настройку окружения командой ./baxter.sh

* rosrun -baxter_tools enable_robot.py -e

* Дальше свои скрипты python my_script.py

* Скрипты должны быть в PYTHONPATH каталоге
